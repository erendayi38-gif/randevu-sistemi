# Randevu / Rezervasyon Sistemi

## 📌 Proje Açıklaması
Bu proje, kullanıcıların belirli bir hizmet için tarih ve saat seçerek randevu oluşturabildiği basit bir web uygulamasıdır.

## 🚀 Özellikler
- Yeni randevu oluşturma
- Tarih ve saat seçimi
- Çakışan randevuları engelleme
- Randevu listeleme
- Randevu iptal etme

## 🛠️ Kullanılan Teknolojiler
- Java
- Spring Boot
- Spring Web
- Thymeleaf
- Maven
